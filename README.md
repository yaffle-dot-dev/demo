# Yaffle Demo

This repository is a standalone demo of a `yaffle.toml` configuration with multiple Terraform workspaces.

> Source of truth: this demo lives in the main Yaffle monorepo. The public repo is a one-way published example of this directory.

## Contents

- `yaffle.toml` showing environments, workspaces, triggers, and approvals
- `infra/` example Terraform workspaces for shared infra, app, database, feature flags, core infra, and sandbox environments

## Purpose

Use this repo as a reference implementation for how a Yaffle-managed infrastructure repository can be structured.
