# Contributing

Thanks for taking a look at the Yaffle demo repo.

## Source and sync model

- Yaffle maintainers primarily work in the main Yaffle monorepo.
- This public repo is a published example repository, not a primary collaboration surface.
- If you spot a problem or want a change, please open an issue so the Yaffle team can update the monorepo source.

## Before suggesting a change

Keep the demo self-contained and readable.

- update `yaffle.toml` and `infra/` together when they need to stay in sync
- avoid adding secrets, credentials, or real infrastructure identifiers
- prefer simple, educational examples over production complexity
